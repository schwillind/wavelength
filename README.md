A set of tools for Amateur Radio, Morse Code, etc

## Prereqs

Node LTS 12.x

## Running Wavelength

```bash
# Clone this repository
git clone https://github.com/schwillind/wavelength.git
# Go into the repository
cd wavelength
# Install wavelength's dependencies
npm install
# Run wavelength
npm start
```
